# AI Face Landmark Tracker

AI Face Landmark Tracker is a web-based application that detects faces and maps facial landmarks from uploaded images using MediaPipe FaceMesh.

## Features
- Upload or drag & drop images
- Detect multiple faces
- Display facial landmark mesh
- Draw bounding boxes around faces
- Smile detection using landmark ratios
- Toggle controls for landmarks, boxes, and smile detection
- Face count display

## Technologies Used
- HTML
- CSS
- JavaScript
- MediaPipe FaceMesh
- Canvas API

## How It Works
The application uses MediaPipe FaceMesh to analyze the uploaded image and detect facial landmarks. These landmarks are then drawn on a canvas along with bounding boxes and smile detection indicators.

## Usage
1. Upload an image.
2. The AI processes the image.
3. Detected faces and landmarks appear on the canvas.
